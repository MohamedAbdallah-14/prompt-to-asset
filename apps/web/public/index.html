The actual pipeline

Producing production assets is a routing + post-processing problem — not a prompt-engineering problem. [research]

  1. Classify — closed enum: logo · app_icon · favicon · og_image · illustration · icon_pack · sticker · transparent_mark · hero.
  2. Route — data-driven model selection. Transparency → gpt-image-1 / Ideogram 3 / Recraft V3. Native SVG → Recraft V3. Never imagen-3 for transparency.
  3. Rewrite — dialect-appropriate. Tag-salad for SD, narrative prose for Flux/Imagen/gpt-image, quoted strings for Ideogram text. Never negative_prompt on Flux (it errors).
  4. Generate — seed-pinned, provider adapter per family.
  5. Validate — checkerboard FFT probe, alpha coverage, moderation ping. Reject + reroute on failure.
  6. Matte — BiRefNet / BRIA RMBG-2.0 / LayerDiffuse / difference matting / native.
  7. Vectorize — Recraft /vectorize · vtracer · potrace. LLM-authored SVG for simple geometry.
  8. Export — iOS AppIcon.appiconset, Android adaptive + monochrome, PWA maskable, favicon bundle, OG via Satori.
  9. Re-validate — tier-0 deterministic + tier-1 metrics + optional tier-2 VLM-as-judge.
  10. Cache — content-addressed by (model, seed, prompt_hash, params_hash). 20–40% hit rate on brand iteration.

11 small tools, not one god-tool

Agents orchestrate; we provide primitives. Every tool prefixed asset_* to avoid Copilot / Cursor collisions.

asset_enhance_prompt

Classify + route + rewrite. Returns AssetSpec JSON. Read-only, idempotent.

asset_generate_logo

PNG master + SVG mark + monochrome variant. Routes by text-length.

asset_generate_app_icon

iOS AppIcon.appiconset + Android adaptive + PWA maskable + visionOS parallax from one master.

asset_generate_favicon

.ico + icon.svg + icon-dark.svg + apple-touch-icon (opaque) + PWA 192/512/maskable + <link> snippet.

asset_generate_og_image

1200×630 via Satori templates. Real fonts. Deterministic. No garbled headlines.

asset_generate_illustration

Brand-locked illustration set. IP-Adapter / LoRA / style_id / Flux.2 refs injected every call.

asset_remove_background

BiRefNet · BRIA RMBG · LayerDiffuse · difference · native. Alpha-aware validation.

asset_vectorize

Raster → SVG via Recraft /vectorize, vtracer, or potrace. Path-count quality signal.

asset_upscale_refine

Asset-type-aware. DAT2 for flat, SUPIR for photo, img2img for polish. Never diffusion on logos.

asset_validate

Tier-0 deterministic · tier-1 metrics · tier-2 VLM. Always runs before the asset is declared ready.

asset_brand_bundle_parse

brand.md / DTCG / AdCP / raw text → canonical BrandBundle schema.
